Private Online Diary

mode of use:

As a user, I want to securely create and store my private diary entries, so I can reflect on my experiences and thoughts in a safe and personal space.

  • Acceptance criteria:
  • I can register and log in with a secure password or social login.
  • I can write and edit diary entries with rich text formatting.
  • Entries are stored securely on the server and protected from unauthorized access.
  • I can manage deletion of individual entries or my entire account.

User::

As a user, I want to organize my entries through tags and dates, so I can easily find and revisit specific memories.

  • Acceptance criteria:
  • I can assign relevant tags to entries.
  • I can search for entries by tags or keywords within the content.
  • I can filter entries by date range or specific calendar year/month.
  • I can view a chronological list of all entries.
